# Vue Electron Sql

> 该脚手架集成了 Vue、Elctron、Sql、Element，方便开发桌面端的 Vue 应用程序

## Build Setup

```bash
# install dependencies
yarn

# serve with hot reload at localhost:9080
yarn run dev

# build electron application for production
yarn run build

# lint all JS/Vue component files in `src/`
yarn run lint

```
